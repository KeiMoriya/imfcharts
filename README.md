
# IMF Charts

## Goal

The goal of this project is to allow easy creation of charts styled for IMF publications.
The benefits of this are:

- Automate generation of charts
- Generate reproducible charts
- Allow easy switching betwen themes so that charts can be reused for different publication types
- Provide the **graphical output** (pdf, png, etc.) as well as **graphical objects** for further customization

The goal is to provide these libraries in the following:

- R
- Stata
- Python
- interactive format (Python/R)
- Matlab

This repository at https://github.com/imfgit/imfcharts is only available for those who have a GitHub organization license.
Documentation should be proivded on the IMF intranet at http://www-intranet.imf.org/fundwide/info/IMFCharts/Pages/default.aspx
and provide detailed documentation, examples and latest updates.
